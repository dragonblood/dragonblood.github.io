    Eridium

    AI AutoDubbing

    Automated AI-assisted dubbing system. A user will upload a short video in a foreign language, after receiving a target video audio is parsed and transcribed, the transcription is then translated using google translation API and converted to speech using Text-to-Speech API. Generated speech is closely matched to the target person's audio using Multi-Tacotron Voice Cloning.

    Github Demo
